## Introduction

**Nmap** (Network Mapper) is a powerful open-source utility designed for network discovery and security auditing. It is widely used by system administrators, security professionals, and penetration testers to analyze networks, identify active devices, and detect open ports and services running on remote hosts. By sending specially crafted packets and analyzing responses, Nmap can determine which systems are online, what operating systems they are running, and which network services are available.

One of the key strengths of Nmap is its flexibility and extensive feature set. It supports multiple scanning techniques such as TCP connect scans, SYN scans, UDP scans, and version detection. These capabilities allow users to perform detailed assessments of network infrastructure while adapting the scan method to different environments and security configurations. Nmap also includes host discovery, OS detection, service version detection, and advanced scripting features.

The Nmap Scripting Engine (NSE) enables automation and deeper analysis by allowing users to run scripts that detect vulnerabilities, misconfigurations, and security issues. This makes the tool useful not only for mapping networks but also for identifying potential weaknesses in systems and services.

Nmap can be used from the command line, making it highly efficient for experienced users, but graphical interfaces such as Zenmap provide a more user-friendly way to visualize scan results. Due to its reliability, extensibility, and strong community support, Nmap remains one of the most essential tools for network security assessment and infrastructure monitoring.
